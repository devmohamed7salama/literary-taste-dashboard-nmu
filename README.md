# 📚 منصة التذوق الأدبي الرقمية | Arabic Literary Taste Dashboard

داشبورد تفاعلية احترافية لمادة "التذوق الأدبي" لطلبة كلية علوم وهندسة الحاسب - جامعة المنصورة الجديدة.

![Dashboard Preview](https://via.placeholder.com/1200x600?text=Arabic+Literary+Taste+Dashboard)

## ✨ المميزات الرئيسية
- **هوية بصرية فاخرة:** تصميم مستوحى من المخطوطات العربية القديمة والورق التراثي.
- **اختبارات تفاعلية:** تحويل بنك الأسئلة (125+ سؤال) إلى نظام كويز تفاعلي مع تصحيح فوري.
- **إحصائيات ذكية:** تتبع معدل الإجابات الصحيحة ونسبة الإنجاز الكلي في المادة.
- **تجربة مستخدم (UX) متطورة:** 
  - Sidebar ثابت للوصول السريع للأقسام.
  - محرك بحث فوري داخل المحتوى.
  - مؤشر تقدم (Progress Bar) علوي.
- **متجاوب بالكامل:** يعمل بكفاءة عالية على جميع الشاشات (Mobile, Tablet, Desktop).

## 🚀 التقنيات المستخدمة
- **React 19**
- **Vite** (للسرعة القصوى في التطوير)
- **Tailwind CSS v4**
- **Framer Motion** (للأنيميشن السلس)
- **Lucide React** (للأيقونات)

## 🛠️ التشغيل المحلي
```bash
# تثبيت التبعيات
npm install

# تشغيل المشروع
npm run dev
```
